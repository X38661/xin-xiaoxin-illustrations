# Xin 小新式淘气小孩正文配图 Skill

这是一个用于中文长文、知识卡片、Notion 文档、方法论内容的正文配图 Skill。

它不是单条插画 prompt，也不是 PPT 模板，而是把“读文章、找画面、生成配图、检查风格”封装成一个可复用流程。

> 说明：这个 Skill 使用的是“受顽皮童真气质启发的原创淘气小孩 IP”，不是官方蜡笔小新同人或复刻角色。实际使用时请保持原创角色设定，避免复制受保护形象。

## 它能做什么

- 先读中文正文，找到真正值得画出来的判断、流程、卡点或隐喻。
- 生成 16:9 横版白底正文配图。
- 使用原创淘气小孩 IP，让角色参与核心动作，而不是站在旁边装饰。
- 保持黑色手绘线稿、少量红橙蓝中文批注、大量留白。
- 避免 PPT 信息图、商业插画、低幼卡通和复杂架构图。

## 适合谁

- 写中文长文的人
- 做知识卡片的人
- 写 Notion 文档的人
- 做方法论内容的人
- 做课程、社群、产品文档的人
- 想把抽象概念画成人话图的人

## 怎么用

把这个目录放到你的 Codex skills 目录下：

```text
~/.codex/skills/xin-xiaoxin-illustrations/
```

然后在 Codex 里调用：

```text
Use $xin-xiaoxin-illustrations to 给这篇文章生成 6 张正文配图。
```

你也可以把 `references/xiaoxin-kid-ip.md` 改成自己的角色设定，比如小猫、小狗、机器人、品牌吉祥物。

关键不是换皮，而是让角色参与核心动作：掉坑、递材料、过桥、接住用户、搬运信息、按按钮、修补流程。
