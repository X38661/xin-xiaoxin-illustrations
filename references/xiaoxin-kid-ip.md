# 原创淘气小孩 IP

## 角色定义

这是 Xin 中文正文配图里的固定视觉 IP：一个受“蜡笔小新式顽皮童真”启发的原创淘气小孩。

借鉴的是气质，不复刻官方角色。可以参考：短小比例、夸张脑袋、厚眉大眼、无辜又欠揍的表情、认真做荒诞事情的反差感。不要直接画官方蜡笔小新，不要复刻红上衣黄短裤、完全一致的脸型、发型、眉眼比例、服装组合或标志姿势。

## 外形

- 原创幼儿角色，短小比例，大脑袋，小身体。
- 黑色手绘线条，轻微抖动，不精致矢量。
- 眉毛可以粗、弯、夸张，但不要完全照搬官方形状。
- 眼睛可以大而圆，也可以豆豆眼，但要保持原创。
- 脸颊和耳朵可以夸张，轮廓要有手绘感。
- 服装使用原创搭配，不使用红上衣 + 黄短裤的官方组合。
- 表情是无辜、困惑、得意、认真或轻微捣蛋，不要卖萌。

## 性格

- 很淘气，但不是幼稚卖萌。
- 看起来不懂，其实正在把复杂系统搞出一个怪办法。
- 有儿童式误读和荒诞行动力。
- 像一个认真执行工作流的小孩，把大人的抽象概念变成可笑但成立的物理动作。
- 画面可以好笑，但核心结构必须清楚。

## 常见职责

让原创淘气小孩承担核心动作：

- 把混乱素材塞进奇怪机器。
- 用玩具铲子挖出关键判断。
- 坐在巨大按钮上触发自动化。
- 把信息线团缠成一条路径。
- 用蜡笔给流程打补丁。
- 把一堆观点装进小书包。
- 在两个断点之间搭积木桥。
- 拿放大镜看常见坑。
- 把输入、处理、输出当成儿童玩具来摆弄。
- 在纸箱、抽屉、门、漏斗、转盘、黑盒旁做荒诞但有效的事。

## 禁止

- 不要直接画官方蜡笔小新。
- 不要使用官方红上衣 + 黄短裤组合。
- 不要复刻官方发型、脸型、眉眼和姿势。
- 不要把角色画成动漫同人、表情包或儿童海报。
- 不要让角色只是站在角落里看。
- 不要让角色抢走结构表达。
- 不要画得太商业、太圆润、太精致。
- 不要做成可爱卡通贴纸。

## 判断标准

如果去掉原创淘气小孩，图的核心隐喻还能完全成立，说明角色太装饰了；要重写提示词，让角色成为动作主体。

如果第一眼像官方蜡笔小新截图或同人，而不是“受顽皮童真启发的原创正文解释图”，说明相似度过高；要换服装、五官比例、发型轮廓和动作。
